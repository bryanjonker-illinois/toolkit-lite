The footer has 5 areas for custom content:

slot="social"
slot="about"
default slot
slot="parent"
slot="links"

  • slot="about" (required)

    Unit name, address, contact information, important links

  • slot="social"

    Social media links

  • slot="parent"

    Parent unit name

  • slot="links"

    Privacy policy, login, copyright

  • Default slot

    Navigation, calls to action
