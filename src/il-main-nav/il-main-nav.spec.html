Main navigation

The main navigation component is used within the standard header and offers a reliable collection of links for navigating the site. These links can be arranged in nested lists to create expandable navigation sections.

Structure

The navigation is composed of items. There are two primary types of items:

  • Links take visitors directly to web pages.
  • Sections are groups of links which can be expanded and collapsed. Sections have labels, and in some sections the label is also a link.
